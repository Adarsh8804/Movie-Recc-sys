# Movie Recommendation System

A Python-based movie recommendation system that recommends movies based on user interests and similar movie titles.

## Features

- User Login Authentication
- Discover Movies Based on Interests
- Find Similar Movies
- SQLite Database Integration
- Terminal-based Interface
- Fast Recommendation Engine

## Technologies Used

- Python
- SQLite3
- Pandas
- NumPy
- Scikit-learn
- Rich

## Project Structure

```bash
Movie-Rec-sys/
│
├── database/
├── src/
│   ├── modules/

│
├── requirements.txt
```

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python src/main.py
```
